Title <マインスイーパー><br><br>

---ゲーム進行---<br>
　行、列、地雷数を入力<br>
　ボードを作成<br>
　地雷を踏むまで採掘、フラグ設置を繰り返す<br>
　採掘した周囲8マスに地雷がない場合、周囲を自動採掘。再帰的に繰り返す。<br><br>
 
---クリア条件---<br>
　地雷以外のすべてのセルを掘り終えるとクリア<br><br>
 
 ---操作方法---<br>
 　左クリック　採掘<br>
 　右クリック　フラグを設置<br>
